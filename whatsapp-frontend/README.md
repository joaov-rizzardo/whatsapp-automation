# whatsapp-frontend

Next.js 16 (App Router) + React 19 web client for the WhatsApp automation project.

```bash
npm install
npm run dev
```

Runs on [http://localhost:3000](http://localhost:3000). Requires the backend (`whatsapp-backend/`) running on `:3333` and PostgreSQL up.

See `CLAUDE.md` for architecture, conventions, and the design system.
